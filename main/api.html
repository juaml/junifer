get_elements

(List) Returns a list of elements that can be grabbed. The elements can be strings, tuples or any object that will be then used as a key to index the datagrabber

__getitem__(element)

(dict[str -> Path]) Returns a dictionary of paths for each type of data required for the specified element. Use the element as a key to index the datagrabber.

class junifer.datagrabber.base.BIDSDataGrabber(types=None, patterns=None, **kwargs)

BIDS DataGrabber class. Implements a DataGrabber that understands BIDS database format.

Attributes
datadir

Returns ——- Path to the data directory.

List of data types to be grabbed.

patternsdict[str -> str]

Patterns for each type of data.

Methods

get_elements: list[str]

Returns a list of elements that can be grabbed. Each element is a subject in the BIDS database.

__getitem__(str): dict[str -> Path]

Returns a dictionary of paths for each type of data required for the specified element. Each occurrence of the string {subject} is replaced by the indexed element

Initialize a BaseDataGrabber object.

Parameters
typeslist of str

The types of data to be grabbed.

patternsdict[str -> str]

Patterns for each type of data. The keys are the types and the values are the patterns. Each occurrence of the string {subject} in the pattern will be replaced by the indexed element.

That directory where the data is/will be stored.

class junifer.datagrabber.base.DataladDataGrabber(rootdir='.', datadir=None, uri=None, **kwargs)

Datalad DataGrabber class. Implements a DataGrabber that gets data from a datalad sibling.

Attributes
datadir

Returns ——- Path to the data directory.

uristr

URI of the datalad sibling.

Methods

install:

Installs (clones) the datalad dataset into the datadir. This method is called automatically when the datagrabber is used within a with statement.

remove:

Remove the datalad dataset from the datadir. This method is called automatically when the datagrabber is used within a with statement.

Initialize a DataladDataGrabber object.

Parameters
rootdirstr or Path

The path within the datalad dataset to the root directory.

datadirstr or Path

That directory where the datalad dataset will be cloned. If None, (default), the datalad dataset will be cloned into a temporary directory.

uristr

URI of the datalad sibling.
